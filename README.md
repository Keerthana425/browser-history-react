
# Browser History App

## 🚀 Live Demo

(https://keerthihistory.ccbp.tech/)

---

## 📌 Description

A React-based browser history application that allows users to search, filter, and delete browsing history items dynamically. The app provides a clean UI and real-time updates using React state management.

---

## 🛠️ Tech Stack

* React JS
* JavaScript (ES6+)
* CSS

---

## ✨ Features

* 🔍 Real-time search functionality
* ❌ Delete individual history items
* ⚡ Dynamic UI updates using state
* 📱 Responsive design

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/Keerthana425/browser-history-react.git
```

2. Navigate to project folder:

```bash
cd browser-history-react
```

3. Install dependencies:

```bash
npm install
```

4. Run the app:

```bash
npm start
```
